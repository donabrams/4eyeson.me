Parchment HTML Template Help

Overview

Thanks for your interest in the Parchment HTML template. This document will help you understand how to customize the template to suit your own needs. The following content outlines the most common tasks that you might want to perform in order to make the Parchment template your own. Please note that this document assumes that you already have HTML and CSS know-how.

File Structure

The Parchment HTML template has a simple file structure that allows the author to quickly find the template's assets. On the root of the web site all the HTML files can be found. Also on the root there's a folder called "assets". Within "assets" there are two more folders labeled "css" and "images" . In the "css" folder you'll find "main.css". "main.css" contains all the styles for the template. In the "images" folder you'll find all the images used in the template. Another folder which is located on the root of the web site is labeled "offline". Any file that has anything to do with the creation of the template, but is not meant to be uploaded to the server can be found in "offline". For example, the Photoshop file labeled "twp-parchment.psd" is in the "offline" folder. Finally a folder labeled "help" can be found on the web site root. The "help" folder contains all the materials for this help file.

Logo

The template can be adjusted to accommodate the size of any logo, but logos that are wide (opposed to tall) are preferred from an aesthetic point of view. After you have prepared you logo, name it "logo.gif" and save it into the "assets/images" folder. If you need to save your logo as a JPEG than you will need to change the source of the logo image within the ".logo" class. The preferred maxium dimensions of your logo are 440 by 90 pixels. The logo dimension are set in the ".logo" class, but do not need to be adjusted unless your logo is larger than 440 by 90 pixels.

Once your logo is in place you should edit the text within the <H1>tag. Applying the <H1> tag to your business name allows search engines to easily index your site and identify your brand. The <H1> tag has the ".hide" class applied to it so it does not overlap your logo graphic.

Global Navigation

Global navigation items can be added, modified or removed to suit your site map. The global navigation is a unordered list of text links. The unordered list is contained with a DIV that has the ".global-nav" class applied to it. You can simply add, modify or remove the text links within the <li> tags.

The upstate and rollover state for the global navigation are controlled by CSS. To customize the global navigation link's upstate edit the ".global-nav a" selector. To customize the rollover state edit the ".global-nav a:hover" selector.

The global navigation allows the author to set a onstate for the current navigation items. So if the user is on the "About" page the "About" link is highlighted. To set the onstate for any global navigation item apply the ".current" class to the appropriate <a> tag. You may customize the onstate style by editing the ".current" class.

Sub Navigation

The Parchment template gives the author the ability to include sub navigation items. The sub navigation appears in the right column of the template. Just like the global navigation, the sub navigation is a unordered list of text links. The <ul> tag which contains the sub navigation items has the ".side-nav" class applied to it. You can add, modify or remove list item links in order to suit your site map.

The upstate and rollover state for the sub navigation are controlled by CSS. To customize the sub navigation link's upstate edit the "ul.side-nav li a" selector. To customize the rollover state edit the "ul.side-nav li a:hover" selector.

Headers

In order to keep you site semantically correct it's best to use the header elements of HTML. I always tend to use the <H1> tag only once for the business name. All other titles are wrapped in the <H2>, <H3>, or <H4> tag. You may customize any style applied to the Header elements by simply editing the proper selector (i.e. h2, h3, or h4).

The ".box-title" class can be applied to any <H3> or <H4> tag to give the title a underlining 3 pixel rule. The ".box-title" class effect can be seen on the right column titles of the interior template page. The ".box-title" class is also applied to the three subtitles found on the home page.

General Elements

Raised Corner Effect

Content groups such as the ones found on the right column of the interior page and home page have a parchment corner that appears to be raised with a shadow. To get that effect apply the ".page-flip" class to containing block element such as a DIV.

List Items

The list items in the main body column are indented with a dashed line on the bottom of each item. If you wish to customize this effect edit the ".main-content ul li" selector for unordered lists or the ".main-content ol li" selector for ordered lists.

Apply the ".line" class to a <ul> tag if you do not want the list items to be intended and you would like a 1 pixel line under each item.

Images

There are three classes for images. All three classes apply a 5 pixel white border and set the proper margins to the image. Apply the ".left" class to a <IMG> tag to place the image to the left side of the containing block. Apply the ".right" class to a <IMG> tag to place the image to the right side of the containing block. Finally, apply the ".side" class to a <IMG> tag if you only want the image to have the border and a bottom margin. The ".side" class is really only needed for images that you place in the side column of the interior page.

Text Links

The global text link style is bold and blue. If no link style is applied otherwise all text links are bold and blue. You can customize the global link style by editing the "a" selector. To customize the global text link rollover effect edit the "a:hover" selector.

All text links found in the side bar, found on the right hand side of the interior pages, are bold and green. The bold effect is from the global text link style cascading. To customize the side bar text links edit the ".side-content a" selector. To customize the side bar text link rollover effect edit ".side-content a:hover".

Summary

There are plenty of ways to customize the Parchment HTML template to better fit your needs. Most of the methods have been explained here, but there is always more that you can do. Use your creativity and have fun building your web project.
